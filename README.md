# ChainMatrix

A physics/puzzle game where players trigger chain reactions to solve levels and achieve high scores.

## Features
- Tap/click to place or trigger objects
- Chain reaction mechanics with cascading interactions
- Multiple levels and increasing difficulty
- Score and star system
